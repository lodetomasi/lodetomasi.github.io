About me

I am a PhD Candidate at the University of L'Aquila, working on AI security and LLM-driven code protection under the supervision of Prof. Antinisca Di Marco, Prof. Phuong T. Nguyen, and Dr. Claudio Di Sipio from the SWEN Group. I also hold a position as Head of Core Platforms at SIAE, where I lead the engineering team responsible for platform modernization and ML/AI infrastructure. I received my MS in Applied Data Science from the University of L'Aquila, focusing on machine learning applications in industrial settings.

My research focuses on the intersection of large language models and software security, particularly developing novel approaches to code obfuscation and protection using generative AI. I am investigating the paradox of LLMs that can identify vulnerabilities with high accuracy yet generate vulnerable code, with applications to secure software engineering practices. Furthermore, I work on scalable ML systems for enterprise environments.

You can find a list of my publications at Google Scholar and ORCID.

Outside of work, I am an avid runner and have completed two marathons with a personal best time of 3:14:56. I find that long-distance running provides the perfect balance to the intellectual challenges of research and engineering.

Research Interests

  • Code Security & Program Obfuscation - Advancing the state-of-the-art in source code protection through information-theoretic approaches and AST-based transformations at scale
  • Formal Methods & Verification - Building provably-secure code transformation systems with semantic preservation guarantees for production environments
  • ML for Code Security - Leveraging neural architectures and adversarial learning to create next-generation code protection mechanisms with measurable security bounds
  • Cryptographic Program Analysis - Developing novel cryptographic models for program transformation with bounded information leakage in distributed systems
  • Theoretical Foundations of Software Security - Establishing fundamental limits and complexity-theoretic bounds for obfuscation-deobfuscation problems in modern software ecosystems
